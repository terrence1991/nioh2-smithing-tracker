# 仁王2 存档解析器使用说明

## ⚠️ 重要提示：PC存档需要解密

你的存档文件 `SAVEDATA.BIN` 是**加密的**！检测到的文件头是 `7F 9C 7F E4...`，这表明需要先解密。

---

## 🔓 如何解密存档

### 方法1：使用 Nioh2SaveEditor (推荐)

1. **下载工具**
   - 访问: https://github.com/alfizari/Nioh-2-Save-Editor
   - 下载最新版本的 `Nioh2SaveEditor.zip`

2. **运行工具** (需要Windows)
   ```bash
   # 在Windows上
   1. 解压下载的文件
   2. 运行 Nioh2SaveEditor.exe
   3. 选择 File -> Open Save File
   4. 选择你的 SAVEDATA.BIN
   5. 工具会自动解密并显示数据
   ```

3. **导出数据**
   - 方法A: 直接从工具中查看已收集的物品
   - 方法B: 如果工具支持，导出为JSON格式

### 方法2：使用已解密的存档

如果你已经有解密后的存档文件：
- 直接上传到网页工具即可

### 方法3：手动查找制作书/武技书

你也可以：
1. 在游戏中查看收集进度
2. 在铁匠铺查看已解锁的制作书
3. 在武道场查看已学习的武技

## 🔧 技术细节

### 存档结构 (解密后)
```
物品起始位置: 0x105EC8
物品大小: 0x88 字节
物品槽位: 900 个

物品ID格式:
- 16位小端序
- 需要字节序转换
```

### 文件识别
```
加密文件特征:
- 文件头: 7F 9C 7F E4 ...
- 数据看起来随机
- 无法直接读取

解密文件特征:
- 文件头: 00 00 00 00 或正常数据
- 可以读取到真实的物品ID
```
